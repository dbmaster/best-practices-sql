Best practices

BP a free SQL Server health check stored procedure that looks for a lot of common health and performance issues. If you just use the default parameters, you’ll get a prioritized list of problems, but by tweaking the input parameters, you can affect how sp_Blitz® runs.

Requirements

SQL Server 2005 or newer. SQL Server 2000 is not supported, nor are databases in 70 compatibility mode.

Sysadmin permissions. checks a lot of system-level diagnostic data.
